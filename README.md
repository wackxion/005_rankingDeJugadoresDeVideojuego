# 005_rankingDeJugadoresDeVideojuego

# ***Proyecto 005: Ranking de Videojuego (Persistent Storage)
Este proyecto consiste en crear un script en Python que permita gestionar una base de datos de jugadores y sus puntajes máximos.
Lo que debe hacer el programa:

- Conexión: Conectarse a un servidor local de MariaDB 
- Registro de Datos: Permitir ingresar el nombre de un jugador y su puntaje.
- Consulta SQL: Mostrar el "Top 5" de los mejores puntajes ordenados de mayor a menor
- Actualización: Si un jugador ya existe, actualizar su puntaje solo si el nuevo es mayor al anterior.
